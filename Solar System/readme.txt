Author: Mitchell Loveall

--------------------------------------------------
| Description:
|
| A real-time rendering project that I made by
| using webGL to simulate solar system, with a
| purpose of practicing skills on rendering 3D
| objects and implementing basic camera controls.
--------------------------------------------------

--------------------------------------------------
| Caution:
|
| The only browser that I had tested on is Google
| Chrome. Planets may not display correctly, or
| have potential bugs, on the other browsers.
--------------------------------------------------

--------------------------------------------------
| Instruction:
|
| Unzip the file, then go to the "web" folder
| inside where you just unzipped everything to,
| there should be a file named 'solarsystem.html'.
| Open this file in Chrome, and here it goes.
--------------------------------------------------

--------------------------------------------------
| Controls:
|
| Drag by using left mouse button to rotate the
| scene; drag by using right mouse button to move
| the camera; rotate middle mouse wheel to zoom
| in or out.
--------------------------------------------------
